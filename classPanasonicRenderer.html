Detailed Description

A static class consisting of two functions that generate IrSignal-s from the Panasonic protocol parameters.

The Panasonic protocol is given in IRP notation as {37k,432}<1,-1|1,-3>(8,-4,2:8,32:8,D:8,S:8,F:8,(D^S^F):8,1,-173)* [D:0..255,S:0..255,F:0..255]
